# Project Overview

## Project Vision
_Purpose of the website:_
- Serve as a central hub for my professional and academic work.
- Showcase my skills and growth as a computer science student.
- Provide a way for people to learn about me and contact me.
- Act as an online resume and portfolio.

## Targeted Audience
_Primary audience:_
- Employers
- Hiring Managers
- Internship providers

_Secondary audience:_
- Computer science students 
- Lecturers and professors
- Academic researchers interested in my work.

_Additional audience:_
- Friends and peers.
- Open-source contributors
- Collaborators
- Anyone interested in my projects and learning journey

## Core Website Sections
_Home page:_
- Includes a professional summary about me.
- my picture.
- A quick navigation panel.

_Projects page:_
- Includes of featured projects and collaborations.
- project details.
- a search bar to look up for projects.
- GitHub link for each project.
- Tech stack used in each project.

_Book Collection page:_
- a collage styled structure of all the books I've read so far.
- a search bar with a genre selector to look up for books.
- And a special section for my favourite book.

_Contact page:_
- includes of all my personal info such as my email, GitHub and LinkedIn.
- Also includes a link to download my resume.

_Certifications page:_
- consists a list of all the certifications I have completed so far on various platforms.
- Also includes credentials.
- a search bar to look up for a specific certification or achievement.

## Interactive Features
- Search
- Filters
- Dark mode
- Resume download

## Success Criteria
_Professional:_
- Employers can quickly understand my skills.
- Projects are easy to browse.
- Contact info is easy to be found.

_Personal:_
- Reflects my personality.
- Easy to maintain.
- Can grow over time.

_Technical:_
- Clean codebase.
- Fast loading.
- Accessible.
- Responsive on all devices.
